# Проект "Посмотри в окно"

Учебный проект по верстке интерактивного видеоплеера с поиском видео по городам и времени суток.

## Технологии

- HTML5
- CSS3
- JavaScript

## Функциональность

- Просмотр видеороликов из разных городов
- Поиск по названию города
- Фильтрация по времени суток (утро, день, ночь)
- Адаптивные карточки с видео
- Кастомные элементы форм
